# GameShell

## Install dependencies

```bash
    pip install keyboard, emoji, pygame
```

## Run the game

```bash
    python3 main.py
```

## Game controls

```bash
    Use 'z','q','s','d' to move the player
    Use 'e' to interact
    Use 'i' to open the inventory
    Use 'm' to open the map
    Press 'x' to quit the game
```
